# H2+ Coulomb-subtraction experiment

This directory contains the frozen protocol, electronic-structure scans,
matched neural fits, controls, sensitivity analyses, and publication figure data
for `2026-07-18-where-coulomb-subtraction-helps.md`.

## Environments

Electronic-structure generation, validation, and all neural fits used arm64
CPython 3.14.6 with NumPy 2.5.0. The quantum scripts additionally require Psi4
1.11 and QCElemental 0.50.4. Final aggregation and plotting used CPython 3.14.4,
NumPy 2.3.4, Matplotlib 3.10.7, and Pillow 12.2.0 with a noninteractive backend.

The recorded hardware was an Apple M1 Pro MacBook Pro with 32 GB RAM. The Psi4
scripts restrict themselves to two threads and 4 GB.

## Rebuild order

Run these commands from the repository root. The first eleven commands use the
chemistry environment; the last uses the plotting environment.

```sh
python calcs/h2plus-coulomb/generate_scan.py
python calcs/h2plus-coulomb/validate_scan.py
python calcs/h2plus-coulomb/fit_scan.py --scan calcs/h2plus-coulomb/h2plus-scan.csv --tag primary
python calcs/h2plus-coulomb/fit_scan.py --input log --tag log-input
python calcs/h2plus-coulomb/fit_scan.py --target-scale common-std --tag common-std
python calcs/h2plus-coulomb/fit_scan.py --hidden 10 --tag hidden-10
python calcs/h2plus-coulomb/fit_scan.py --hidden 20 --tag hidden-20
python calcs/h2plus-coulomb/fit_scan.py --selection constant --tag constant-count
python calcs/h2plus-coulomb/generate_scan.py --basis aug-cc-pvqz --output-prefix h2plus-scan-qz
python calcs/h2plus-coulomb/fit_scan.py --scan calcs/h2plus-coulomb/h2plus-scan-qz.csv --tag qz-basis
python calcs/h2plus-coulomb/fit_scan.py --steps 40000 --learning-rate-start 3e-4 --learning-rate-end 3e-6 --tag lower-lr-long
MPLBACKEND=Agg MPLCONFIGDIR=/tmp/mpl-h2plus python calcs/h2plus-coulomb/analyze_results.py
```

The archive includes the raw 5Z and QZ scans, fold assignments, per-seed metrics,
training summaries, and JSON aggregates. Per-geometry out-of-fold prediction
tables are omitted to keep the download small; every one is regenerated by
`fit_scan.py` before its metrics are written.
